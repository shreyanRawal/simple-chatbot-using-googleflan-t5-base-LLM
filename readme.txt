# Simple Flan-T5 Chatbot

A lightweight chatbot built using **Google's Flan-T5 Small model**.  
This project demonstrates how to interact with a large language model (LLM) using Python and the Hugging Face Transformers library.



## Features

- Chat with a text-based AI in your terminal.
- Uses **PyTorch** as the backend.
- Simple setup with only 3 required packages: `torch`, `transformers`, `sentencepiece`.
- Quick to run without fine-tuning or large datasets.



## Requirements

- Python 3.11 (64-bit recommended)
- Packages listed in `requirements.txt`
